&nbsp;   Note over Browser: The default form submission is prevented, so the page does NOT reload and no traditional GET request is made.

&nbsp;   Browser->>Browser: Create note object (content + new Date())

&nbsp;   Note over Browser: The note object is created using e.target.elements\[0].value and the current date.

&nbsp;   Browser->>Browser: notes.push(note)

&nbsp;   Note over Browser: The new note is added to the local notes array stored in browser memory.

&nbsp;   Browser->>Browser: redrawNotes()

&nbsp;   Note over Browser: The DOM is updated immediately by re-rendering the notes list.

&nbsp;   Browser->>Server: HHTP POST /exampleapp/new\_note\_spa (application/json)

&nbsp;   Note over Browser: The note is sent asynchronously to the server using XMLHttpRequest.

&nbsp;   Server-->>Browser: 201 Created

&nbsp;   Note over Server: The server stores the note and responds with status 201 (Created).
